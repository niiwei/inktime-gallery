# InkTime Gallery Brand System

InkTime Gallery 的视觉身份以“夕阳记忆窗口”为核心：圆形窗口代表回看照片的入口，夕阳金色代表温暖的记忆与自动轮换的壁纸。

## Tokens

- Accent: `#0d7a55` in light mode, `#71d9ad` in dark mode.
- Logo accent: sunset gold `#ffd86d`, amber `#f79b2f`, reflection orange `#ea7d31`.
- Shape: soft macOS-style rounded rectangles, large surfaces use 22px, controls use pill or 10px.
- Material: translucent surfaces with solid fallbacks for reduced transparency.
- Typography: Apple system stack, keeping the app close to native macOS.
